# Laporan Tugas

## Poin-Poin perubahan yang dilakukan:

* Penyesuaian build
    * mengubah nama file output eksekusi yang semula **myapp** menjadi **MyApp**
    * menambahkan file direktori /obj untuk menyimpan hasil kompilasi compiler
  
* Perubahan logika program
    * menambahkan variabel **user_input** untuk menyimpan input dari pengguna
    * perubahan output dari yang semula **Hello World** menjadi **Hello <user_input>**
  
* Hasil output sesuai dengan output yang diminta
